# User Management API

Production-ready HTTP API для управления пользователями с системой задач, поинтов и рефералок.

## Технический стек

- Go 1.22+
- PostgreSQL + pgx driver
- JWT Authentication
- Chi Router
- Docker & Docker Compose
- Clean Architecture

## Возможности

-  JWT авторизация
-  Управление пользователями
-  Система задач с наградами
-  Таблица лидеров
-  Система баллов и транзакций
-  Реферальная программа
